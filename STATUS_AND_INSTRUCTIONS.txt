================================================
📞 TROUBLESHOOTING
================================================

"Cannot connect to API"
→ Assurez-vous que Flask est en cours d'exécution
→ Vérifiez que le port 5000 est accessible
→ Restart Flask avec : python app.py

"Port 8501 already in use"
→ streamlit run interface.py --server.port 8502

"Permission denied" (Streamlit)
→ pip install --upgrade streamlit
→ Relancer : streamlit run interface.py

Voir STREAMLIT_GUIDE.txt pour plus de détails
